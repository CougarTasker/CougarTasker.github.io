Towers of Hanoi

Here is a program that I wrote to demonstrate and solve the Towers of Hanoi problem. use the slider to change the number of blocks and press the start button to start the animation

The rules of the game:
  • The tower must be moved from one peg to another
  • Only one block can be moved at a time
  • A block can only be placed ontop of a larger block
